# Agentic_AI_Building_with_Antigravity

I enrolled in an agentic ai coding program hosted by geeks for geeks and sponsored by Google. 
During the program Google's Antigravity AI models were prompted to generate a website of our own preference, which is basically what's called vibe coding. Yeah we vibe coded a fully funcional website.
And I'd say it was a wonderful experience watching the AI build a whole functional website in a matter of minutes.


Within this repository are some of the codes of the website and its features generated by Antigravity.
